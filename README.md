# OreBreaker reborn
This is a new and heavily reworked version of the classic OreBreaker.
This version uses modern technologies such as React, TypeScript, and Zustand.
These choices make the project significantly more convenient for scaling and further development.

**This version is currently in Alpha testing, so the current code is intended for GitHub only.**
